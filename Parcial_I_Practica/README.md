Breve explicación de cómo funciona el programa:
Este programa crea tres nodos usando memoria dinámica con new.
Cada nodo guarda un número que el usuario ingresa.
Después, los nodos se conectan entre sí usando punteros, formando una pequeña lista enlazada.
Luego se recorren uno por uno para mostrar los valores guardados.
Al final, se usa delete para liberar la memoria y evitar que quede ocupada innecesariamente.
